# Onboarding Overview

To create and customize z/OS demo images for TechZone, access to the z/OS provisioning infrastructure is required. This infrastructure uses Ansible Automation Platform to automate the deployment of z/OS instances on IBM Z hardware in the Worldwide Systems Center. Access is available to anyone in the IBM Z Ecosystem organization.

The platform provides capabilities to:
- Provision z/OS instances on-demand for development and testing
- Customize z/OS images with middleware (CICS, Db2, IMS, MQ)
- Create snapshots of configured systems
- Publish finalized demo images to TechZone for technical sales use

Access requires three components: automation tooling (AAP), network connectivity (VPN), and authentication credentials (SSH keys). Follow the three steps in the sidebar to complete onboarding.

**Timeline**: 3-5 business days
