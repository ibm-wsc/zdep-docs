# Demo Delivery

This section contains documentation for technical sales professionals who use WSC z/OS demo images to deliver customer demonstrations and hands-on labs.

## Overview

WSC z/OS demo images are fully-configured, ready-to-use z/OS environments available through IBM TechZone. Each image includes:
- Pre-configured z/OS base system
- Installed and configured middleware (CICS, Db2, IMS, or MQ)
- Sample applications and data
- Documentation and demo scenarios

## Available Demo Images

Browse the catalog of available images:
- **CICS Demo Image** - Transaction processing with sample applications
- **Db2 Demo Image** - Database management and SQL demonstrations
- **IMS Demo Image** - Hierarchical database and transaction management
- **MQ Demo Image** - Message queuing and integration scenarios

## Quick Start

To get started with a demo image:

1. **Request Access** - Ensure TechZone access and WSC VPN credentials are in place
2. **Provision** - Reserve an image through TechZone
3. **Connect** - Use VPN and 3270 emulator to access the instance
4. **Demo** - Follow provided scenarios or create custom demonstrations

## Key Features

All demo images include:
- ✅ **No Setup Required** - Fully configured and ready to use
- ✅ **Sample Applications** - Pre-loaded demo programs and data
- ✅ **Documentation** - Step-by-step guides and scenarios
- ✅ **Flexible Duration** - Reserve for hours, days, or weeks
- ✅ **On-Demand Access** - Available 24/7 through TechZone
