# Provisioning from TechZone

Learn how to provision WSC z/OS demo images from IBM TechZone.

## Overview

IBM TechZone provides on-demand access to pre-configured z/OS demo environments. Each image is fully configured and ready to use for demonstrations and hands-on labs.

## Provisioning Steps

### 1. Access TechZone

Navigate to IBM TechZone and search for "WSC z/OS Demo Images"

### 2. Select Your Image

Choose from available demo images:
- **CICS Demo Image** - Transaction processing demonstrations
- **Db2 Demo Image** - Database management scenarios
- **IMS Demo Image** - Hierarchical database demos
- **MQ Demo Image** - Message queuing integration

### 3. Configure Your Reservation

- **Duration**: Select how long the environment is needed (hours, days, or weeks)
- **Purpose**: Specify the use case (demo, training, POC, etc.)
- **Geography**: Choose the closest data center for optimal performance

### 4. Submit Request

Review the configuration and submit the provisioning request. An email notification is sent when the environment is ready (typically within 15-30 minutes).

## What You'll Receive

Once provisioned, the following information is provided:
- z/OS IP address
- VPN connection details
- Access credentials
- Quick start guide specific to the image
