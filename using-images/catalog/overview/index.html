#Provisioning Process

All images can be provisioned directly from TechZone:

  1. Browse the TechZone catalog
  2. Select the desired image
  3. Complete the reservation form
  4. Receive connection details via email
  5. Connect using VPN and SSH

For detailed provisioning instructions, see the Provisioning from TechZone guide.
