#CICS Configuration Details

#CMCI API

The CICS Management Client Interface (CMCI) provides a REST API for managing CICS resources.

Base URL:

http://<zos-ip>:10080/CICSSystemManagement

Authentication: Basic — user CICSUSER

EndpointDescription
/CICSRegion/CICS62Region information
/CICSDefinitionProgram/CICS62Program definitions
/CICSDefinitionTransaction/CICS62Transaction definitions
/CICSDefinitionFile/CICS62File definitions
/CICSDefinitionBundle/CICS62Bundle definitions

#Security

#CICS SVC

  • SVC number: 216 (DFHCSVCU / DFHCSVC)
  • Installed in: SYS1.PARMLIB(IEASVC00) and SYS1.PARMLIB(LPALST00)

#Liberty Angel

  • Procedure: BBGZANGL in SYS1.PROCLIB
  • Angel name: CICSANGL
  • User: BBGZANGL (UID 9999)

#SSL/TLS

  • Keyring: CICSRING (owner: CICSPRD)
  • Certificate: CICSCERT — self-signed RSA 2048-bit

#RACF Profiles

  • BBG.ANGEL.CICSANGL
  • BBG.AUTHMOD.BBGZSAFM.* (SAFCRED, PRODMGR, ZOSAIO, ZOSWLM, ZOSDUMP)
  • BBG.SECPFX.*
