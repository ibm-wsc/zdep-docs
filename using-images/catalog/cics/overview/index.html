#z/OS Standard + CICS

The z/OS Standard + CICS image is a fully configured z/OS environment with CICS Transaction Server 6.2. It is pre-configured with sample programs, transactions, VSAM files, and Java bundles, and is ready for CICS-based demonstrations immediately after provisioning from TechZone.

#What's Included

  • z/OS 3.1 — See Base z/OS Configuration for the full system layout
  • CICS Transaction Server 6.2 — Configured with CICSPlex SM (CMCI)
  • Sample COBOL program — HELLO program / HELO transaction
  • Java bundles — TERMINAL and VSAM OSGi bundles deployed via CMCI
  • VSAM file — CICS.EXAMPLE.DATA pre-defined and accessible
  • SSL/TLS security — Keyring CICSRING with self-signed certificates

#Quick Start

For VPN, SSH, and 3270 terminal setup, see Connecting to Your Instance.

Once connected, run the HELO transaction to verify CICS is working:

  1. Connect via 3270 emulator to port 2023
  2. Log in: TSO ZOSADMN
  3. Type: HELO — the Hello World message from the HELLO COBOL program appears
