#Machine Learning for z/OS

Click here to go to the Collection on TechZone. From there you can provision an instance under the Environments tab and find documentation under the Resources tab.

#Overview

This collection provides hands-on experience with Machine Learning for z/OS, showing how Machine Learning for z/OS enables infusing AI within transactional applications on IBM z/OS.

Using this environment, you will walk through a guided experience, providing instructions for creating a Machine Learning model for credit card fraud detection and deploying it on z/OS. The following tasks will be completed:

  • AI Model training in JupyterLab
  • Model importation into Machine Learning for z/OS (MLz)
  • Model deployment to an MLz scoring server integrated in a CICS region
  • AI integration with a front-end demo application
  • AI Inferencing within a CICS COBOL transaction
