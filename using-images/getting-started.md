# Getting Started with Demo Images

This guide provides information for getting started with WSC z/OS demo images from TechZone.

## Prerequisites

Before beginning, ensure the following are available:
- Access to IBM TechZone
- WSC VPN credentials
- A 3270 terminal emulator (e.g., IBM Host On-Demand)

## Quick Start

1. **Request Access** - Ensure the necessary permissions are in place
2. **Provision** - Reserve an image through TechZone
3. **Connect** - Use VPN and terminal emulator to access
4. **Demo** - Follow provided scenarios or create custom demonstrations
